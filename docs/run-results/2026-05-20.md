## Conclusion

- Native LoRA remains the failure path for this adapter shape on
  `sglang==0.5.12`.
- Merge-and-serve avoids the native LoRA adapter loader.
- Triton attention serves the merged artifact and completes the chat scoring
  run.
- FlashInfer reaches readiness but fails during paged prefill on the merged
  artifact.
- This tiny adapter did not improve the ticket-triage exact-match metric; the
  fixture is used to reproduce serving behavior, not to train a useful support
  model.
